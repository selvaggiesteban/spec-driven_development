##### 5.13.1.7 Roles and Responsibilities

**DevOps Engineer / SysAdmin**:
- Reviews virtual hosts configuration
- Validates MPM and connection limits
- Verifies logs and rotation
- Checks deployment process

**Security Analyst**:
- Audits security headers (X-Frame-Options, CSP)
- Verifies SSL/TLS configuration
- Reviews file and directory permissions
- Checks that mod_security is configured (if applicable)

**Performance Engineer**:
- Reviews cache configuration
- Verifies compression (mod_deflate)
- Evaluates Keep-Alive usage
- Checks performance under load
