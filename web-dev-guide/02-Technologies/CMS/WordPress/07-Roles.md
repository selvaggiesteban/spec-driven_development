##### 5.6.3.7 Roles and Responsibilities

- [ ] WordPress code follows community standards (validated by Senior WordPress Developer)
- [ ] Security was audited with Wordfence or similar (validated by Security Analyst)
- [ ] Plugin/theme was tested in multiple environments (validated by QA Engineer)
- [ ] Queries are optimized and don't cause slowness (validated by Performance Engineer with Query Monitor)
- [ ] On-page SEO is correctly implemented if it's a theme (validated by SEO Specialist)
