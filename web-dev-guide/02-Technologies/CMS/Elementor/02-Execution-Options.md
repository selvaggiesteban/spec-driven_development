##### 5.7.1.2 Execution Guidelines

- Before starting, make a copy of the original JSON.
- Your only task is to modify EXCLUSIVELY the VISIBLE TEXT displayed on the web page.
- You must edit **only** textual content fields, including: `settings.title`, `settings.editor` (paragraphs, HTML, rich text), `settings.text` (buttons), `settings.items[].item_title` (accordions, tabs, dynamic lists), `settings.items[].item_content` if it exists, any other field within `settings` that contains readable text for the end user, widgets of type `heading`, `text-editor`, `button` that contain visible text.
