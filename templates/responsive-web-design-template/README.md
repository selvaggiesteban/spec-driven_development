# Responsive Web Design Template

A professional, high-performance responsive web design template featuring a contact form with ReCaptcha verification and WhatsApp integration.

## Features

- **Responsive Design**: Fully adaptable to mobile, tablet, and desktop views.
- **Contact Form**: Secure PHP-based contact form with Google ReCaptcha v2/v3 support.
- **WhatsApp Integration**: Floating WhatsApp button for direct communication.
- **Smooth Navigation**: Scroll-aware navigation menu with dynamic styling.
- **Clean Architecture**: English logic with Spanish documentation/comments.

## Setup

1. Clone the repository.
2. Copy `.env.template` to `.env` and fill in your credentials:
   ```env
   CONTACT_EMAIL=your-email@example.com
   RECAPTCHA_SECRET=your-recaptcha-secret
   ```
3. Deploy to a PHP-enabled server.

## Technologies Used

- HTML5 / CSS3 (Normalize.css)
- JavaScript (jQuery, Vanilla JS)
- PHP (Contact form processing)
- Google ReCaptcha API
