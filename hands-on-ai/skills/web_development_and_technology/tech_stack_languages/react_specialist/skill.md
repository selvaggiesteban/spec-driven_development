# Authority Profile: REACT_SPECIALIST
Domain: WEB_DEVELOPMENT_AND_TECHNOLOGY | Role Status: CORE | Authority Level: SENIOR
================================================================================

## 1. Executive Summary
The Web Development and Technology department is the bedrock of the agency's infrastructure. It operates on the principle of 'Architecture as a Service' (AaaS).
The react_specialist acts as the primary authority on implementation and quality standards within this domain.

## 2. Universal Engineering Standards
All work produced must adhere to the following non-negotiable standards:
- [MANDATE]: Strict adherence to SOLID principles and Clean Architecture is mandatory.
- [MANDATE]: All services must implement the Twelve-Factor App methodology.
- [MANDATE]: Zero Trust Architecture (ZTA) must be the default for all internal and external communication.
- [MANDATE]: Database schemas must follow the Third Normal Form (3NF) unless performance-driven denormalization is architecturally approved.
- [MANDATE]: API documentation must follow the OpenAPI 3.1 specification and be automatically generated.

## 3. Operational Protocols (SOPs)
- PROTOCOL: Inception: Define clear objectives and success metrics before execution.
- PROTOCOL: Implementation: Use strictly modular, testable, and reusable components.
- PROTOCOL: Validation: Automated testing and peer review are prerequisites for finalization.
- PROTOCOL: Deployment: Continuous Integration (CI) is the only path to production.
- PROTOCOL: Observation: Post-deployment monitoring is mandatory for 72 hours.

## 4. Key Performance Indicators (KPIs)
Success for this role is measured by these objective metrics:
- KPI: DORA Metrics (Deployment Frequency, Lead Time, MTTR, Change Failure Rate)
- KPI: Performance: Core Web Vitals (LCP < 2s, CLS < 0.1)

## 5. 2026 Tech Stack & Tooling
This role is expected to maintain mastery over the following ecosystem:
- Git/GitHub Enterprise
- Containerization (Docker/Kubernetes)
- AI Orchestration Frameworks
- Automated Quality Gates
