##### 5.9.3.5 Dependencies

- **Authorization Server**: Entity that issues access tokens and manages user permissions. Can be an external or internal IdP.
- **Resource Server**: API or service that protects its resources and exposes them through access tokens.
- **OAuth Client**: Application that requests access to protected resources on behalf of the user.
- **RFC 6749 and 6750 Standards**: Define the roles, flows, and use of OAuth 2.0 bearer tokens.
- **OpenID Connect (OIDC)**: Identity layer built on top of OAuth 2.0 for user authentication.
