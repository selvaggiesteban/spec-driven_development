# Planning as Code
![Status](https://img.shields.io/badge/Status-Operational-green)
![Author](https://img.shields.io/badge/Author-Esteban_Selvaggi-blue)

## Description
A methodology and set of tools to treat project planning and architecture design as source code. This allows for version control, automated validation, and seamless collaboration between human architects and AI agents.

## Author
Esteban Selvaggi ([selvaggiesteban.dev](https://selvaggiesteban.dev))

## Technical Requirements
- JSON Schema for plan validation.
- Markdown for documentation.

## Setup & Installation
1. Clone the repository.
2. Explore the templates in the `schemas/` directory.

## Usage
Define your project roadmap using the provided JSON structures to ensure consistency and machine-readability.

## Technical Recommendations
- Use YAML for complex multi-stage plans for better readability before converting to JSON.
