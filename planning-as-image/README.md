# Planning as Image
![Status](https://img.shields.io/badge/Status-Operational-green)
![Author](https://img.shields.io/badge/Author-Esteban_Selvaggi-blue)

## Description
Visual planning framework that uses image-based representations to communicate complex system architectures and project flows to both humans and vision-capable AI models.

## Author
Esteban Selvaggi ([selvaggiesteban.dev](https://selvaggiesteban.dev))

## Technical Requirements
- SVG/Mermaid.js for diagram generation.
- Vision AI models (GPT-4o) for plan interpretation.

## Setup & Installation
1. Install Mermaid CLI: `npm install -g @mermaid-js/mermaid-cli`.

## Usage
Create `.mmd` files and generate visual maps of your development strategy.

## Technical Recommendations
- Keep diagrams modular to avoid visual clutter.
