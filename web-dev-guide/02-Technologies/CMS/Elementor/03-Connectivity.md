##### 5.7.1.3 Restrictions

**You must NOT modify ANY other JSON property. Especially DO NOT touch:**
- `id`, `elType`, `widgetType`, `elements`
- Array order or structure
- Design properties: `padding`, `margin`, `width`, `content_width`, `flex_*`, `background_*`, `shape_divider_*`, `box_shadow_*`, `typography_*`, `color`, `align`, `line_height`
- Animations (`animation`, `animation_duration`)
- Global styles (`__globals__`)
- Dynamic fields (`__dynamic__`)
- URLs of images, galleries or slideshows (NO need to modify them)
- Any field whose value is NOT clearly visible text on the page
