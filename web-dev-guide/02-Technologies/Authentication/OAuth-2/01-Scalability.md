##### 5.9.3.1 Scalability

- OAuth 2.0 is a highly scalable framework, as it delegates authentication and authorization to dedicated identity providers.
- Allows clients (applications) to operate without storing sensitive user credentials, improving security and reducing attack surface.
- The separation of concerns between the client, resource server, and authorization server facilitates independent scaling of each component.
- Issuing short-lived access tokens and refresh tokens can optimize performance and session management.
