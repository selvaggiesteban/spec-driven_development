##### 5.9.5.7 Roles and Responsibilities

**Backend Developer**:
- Implements the IMAP connection and email processing logic.
- Designs the schema for handling and storing extracted information.
- Ensures secure management of mailbox access credentials.

**DevOps Engineer**:
- Configures firewall and network rules to allow IMAP connection.
- Monitors the performance and availability of the integration.
- Manages credentials securely (e.g., using secrets).

**System/Email Administrator**:
- Provides IMAP mailbox access credentials.
- Configures the email server to allow connections from the application's IP.
- Monitors mailbox usage by the application.
