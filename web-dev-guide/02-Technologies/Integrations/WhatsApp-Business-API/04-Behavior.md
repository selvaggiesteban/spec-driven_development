##### 5.9.9.4 Behavior

- **Message Sending**: The application sends messages to users through WhatsApp API endpoints, using pre-approved templates or session messages.
- **Message Reception**: User messages are delivered to the application through configured Webhooks.
- **Session Management**: User-initiated conversations have a 24-hour window during which free-form messages can be sent.
- **Status Notifications**: Webhooks also report on delivery and read status of sent messages.
- **Template Approval**: Message templates must be submitted to WhatsApp for approval before they can be used for outgoing communications.
