# 🚀 Ultimate n8n AI & Automation Library

Welcome to the most comprehensive collection of production-ready **n8n workflows**. This repository contains over 200+ advanced automations, AI agents, and RAG implementations designed to supercharge your productivity and business operations.

![n8n banner](https://n8n.io/images/og-image.png)

## 🌟 What's Inside?

This library is a curated set of `.json` (saved as `.txt`) workflow exports that you can directly import into your n8n instance. From local LLMs with Ollama to complex RAG systems with Supabase and Qdrant, this repo covers the entire automation spectrum.

## 🚀 How to Use These Workflows

1.  **Find a Workflow:** Browse the file list above.
2.  **Copy the Content:** Open any `.txt` file and copy the entire JSON content.
3.  **Import to n8n:**
    *   Open your n8n editor.
    *   Press `Ctrl + V` (Windows/Linux) or `Cmd + V` (Mac) to paste the workflow directly.
    *   *Alternative:* Use the "Import from File" option in the n8n menu.
4.  **Configure Credentials:** Ensure you set up the required API keys (OpenAI, Google, etc.) in your n8n credentials manager.
