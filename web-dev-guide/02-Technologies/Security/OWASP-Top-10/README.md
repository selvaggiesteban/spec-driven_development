#### 5.23.9. General Security Checklist (Cross-cutting)

- [ ] Defense-in-depth principle applied
- [ ] Least privilege principle at all levels
- [ ] Environment separation (dev/staging/prod)
- [ ] Secrets rotation implemented
- [ ] Backup and disaster recovery tested
- [ ] Security documentation updated
- [ ] Security training for the team
- [ ] Vulnerability disclosure process defined
- [ ] Bug bounty program considered (for critical apps)
