##### 5.9.9.5 Dependencies

- **WhatsApp Business Account**: Configured and verified business profile.
- **Registered Phone Number**: A dedicated phone number for the WhatsApp Business API.
- **Permanent Access Token**: To authenticate API requests.
- **Facebook Business Platform / Meta for Developers**: To manage the application, Webhooks, and phone number.
- **Business Solution Provider (BSP)**: Optional, but recommended to handle API infrastructure (e.g., 360dialog, Twilio).
- **Client Library (SDK)**: Specific libraries to interact with the API (e.g., Meta's official, or BSPs').
