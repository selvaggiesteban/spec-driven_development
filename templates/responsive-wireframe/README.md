# Responsive Wireframe Template

A clean and minimal wireframing template for rapid web prototyping. Focus on structure and user flow without getting distracted by visual design.

## Features

- **Grid System**: Responsive grid for easy layout planning.
- **UI Components**: Placeholder styles for common web elements.
- **Lightweight**: Zero dependencies, just pure HTML/CSS.
