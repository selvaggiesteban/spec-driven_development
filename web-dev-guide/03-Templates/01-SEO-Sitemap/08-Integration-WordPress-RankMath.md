### 6.8. WordPress + Rank Math (CMS and On-site SEO)

- The site uses a child theme if code customizations have been made.
- There are no redundant or duplicate plugins that provide the same functionality.
- WordPress plugins and core are updated to secure versions.
- Core or plugin files have not been directly modified without documentation.
- Rank Math is configured with consistent SEO titles and meta descriptions per content type.
- The sitemap generated by Rank Math/WordPress is accessible and correctly linked.
- There are no important contents with noindex or mistakenly blocked in robots.txt.
