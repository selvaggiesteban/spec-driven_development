== Description ==

**SEO Article Generator Pro** es un plugin completo para WordPress que utiliza la API de Google Gemini para generar artículos optimizados para SEO.

= Características principales =

* **Generación de Artículos con IA**: Crea contenido SEO de alta calidad usando Gemini 2.5 Flash
* **Múltiples Tipos de Contenido**: Guías, tutoriales, listas, comparativas, casos de estudio
* **Optimización SEO Automática**: Meta títulos, descripciones, estructura de headings
* **Generación de Imágenes**: Crea imágenes destacadas con IA
* **Análisis de Contenido**: Escanea tu sitio para detectar oportunidades SEO
* **Enlazado Inteligente**: Sugerencias de enlaces internos basadas en IA
* **Programación de Publicaciones**: Cola de publicación con calendario visual
* **Integración RankMath**: Sincroniza automáticamente con RankMath SEO

= Requisitos =

* WordPress 6.0 o superior
* PHP 7.4 o superior
* API Key de Google Gemini (gratis en [Google AI Studio](https://aistudio.google.com/))

== Installation ==

1. Sube el plugin a la carpeta `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' de WordPress
3. Ve a 'SEO Generator' > 'Configuración'
4. Introduce tu API Key de Google Gemini
5. ¡Comienza a generar artículos!

== Frequently Asked Questions ==

= ¿Cómo obtengo una API Key de Gemini? =

1. Ve a [Google AI Studio](https://aistudio.google.com/)
2. Inicia sesión con tu cuenta de Google
3. Haz clic en "Get API Key"
4. Copia la clave y pégala en la configuración del plugin

= ¿Cuánto cuesta usar la API de Gemini? =

Google ofrece un tier gratuito generoso. Consulta los [precios actuales](https://ai.google.dev/pricing).

= ¿Funciona con Yoast SEO? =

Actualmente la integración nativa es con RankMath. Yoast puede añadirse en futuras versiones.
